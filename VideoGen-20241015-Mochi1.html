Mochi 1: Genmo 开源视频生成模型

📌 概要

Mochi 1 是 Genmo 于 2024 年 10 月发布的开源视频生成模型，在初步评测中展示了高保真运动和强 prompt 遵循能力。该模型"显著缩小了闭源和开源视频生成系统之间的差距"，以 Apache 2.0 许可证发布。

核心价值：在当时是最佳的开源视频生成模型之一，支持 ComfyUI 原生集成和消费级 GPU 运行（通过 CPU offload）。2024 年 11 月新增 LoRA 微调支持。

🏗️ 模型架构

基于 GitHub README 和 Genmo 博客公开的信息：

  • 非对称扩散 Transformer (Asymmetric Diffusion DiT)：Genmo 的核心架构创新
  • 潜在视频扩散：在压缩的潜在空间进行扩散
  • 大规模参数：约 10B 参数级别
  • Flow Matching 目标：使用 flow matching 训练目标替代传统 ε-prediction
  • 多分辨率：支持多种分辨率和宽高比

非对称架构设计

非对称设计允许模型在处理空间和时间维度时使用不同的计算分配。空间维度（单帧）可以使用更多计算资源，而时间维度（帧间关系）使用更轻量的计算。这种设计在保持质量的同时提高了效率。

📊 技术影响

Mochi 1 在学术研究中被广泛引用作为基线和加速目标：

  • CalibAtt (arXiv: 2603.05503)：在 Mochi 1 上实现 1.58x 端到端加速
  • 被多项稀疏注意力加速工作作为测试模型
  • 与 Wan 2.1 14B、CogVideoX 并列为开源视频生成的代表性模型

💡 个人分析

  • 开源先锋：Mochi 1 是 2024 年最重要的开源视频生成模型之一，与 CogVideoX、HunyuanVideo 并列
  • 非对称设计创新：非对称 DiT 架构是效率导向的创新，影响了后续模型设计
  • 生态友好：Apache 2.0 + ComfyUI + LoRA + Gradio，完整的开源生态
  • 竞争定位变化：2025 年 Wan 2.1 发布后，Mochi 1 的性能优势被超越，但其开源贡献价值持久
  • 加速研究价值：作为开源模型被多项加速工作（CalibAtt、AdaCluster）用于验证，说明其在社区中的标准地位
  • Genmo 路线：Genmo 专注于开源模型，与快手 Kling、Runway 等闭源路线形成对比

📐 算法框图

Mochi 1 架构：10B 参数，非对称 DiT，Apache 2.0 开源

Pipeline：
  • 文本编码：T5 编码器
  • 视频 VAE：视频 VAE 编码/解码
  • 非对称 DiT：编码器和解码器不同深度，更高效
  • Flow Matching：连续时间生成框架

💡 核心创新点

创新 1：非对称 DiT — 编码器和解码器使用不同深度，更高效的计算分配。
创新 2：Apache 2.0 完全开源 — 10B 模型完全开源，作为研究基线。
创新 3：研究导向定位 — Genmo 以研究透明度为核心，代码+权重+训练细节全开放。
