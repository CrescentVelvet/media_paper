Real-ISR 技术演进调研：从 ESRGAN 到 OSEDiff 及未来方向

📅 2018-2026 🔬 GAN → Diffusion → One-Step 📄 7+ 篇核心论文 📅 2026-06-29 整理

📋 概述

Real-ISR Super-Resolution GAN Diffusion One-Step Blind SR

Real-ISR（Real-world Image Super-Resolution）旨在从未知、复杂的真实世界退化过程中恢复高质量图像。与早期假设简单双三次下采样的超分方法不同，Real-ISR 需要处理模糊、噪声、JPEG 压缩、传感器噪声等多种退化叠加的复杂情况，是计算机视觉中一个重要的开放问题。

本调研覆盖从 2018 年到 2026 年的 Real-ISR 技术演进，梳理了从 GAN 范式（ESRGAN → BSRGAN → Real-ESRGAN）到 扩散模型范式（StableSR → DiffBIR → SeeSR）再到 单步扩散（OSEDiff）及最新方法（TEASR、SANA-SR、ODTSR、FRAMER）的完整技术路线。

核心演进脉络：① 退化建模从简单到复杂（双三次 → 高阶退化 → 随机打乱）→ ② 生成先验从 GAN 到预训练 T2I 扩散模型 → ③ 推理效率从多步（10-200 步）到单步 → ④ 语义理解从无到有（DAPE 提示提取器）→ ⑤ 架构从 U-Net 到 DiT。

🔬 各方法详解

1. ESRGAN (2018) — Enhanced Super-Resolution Generative Adversarial Networks

GAN RRDB Relativistic Discriminator Perceptual Loss

核心思想：在 SRGAN 的基础上，从网络架构、对抗损失和感知损失三个方面进行全面改进，生成更真实、自然的纹理。

架构设计：

  • RRDB (Residual-in-Residual Dense Block)：移除 Batch Normalization，采用残差密集块嵌套结构。无 BN 设计减少了伪影，密集连接增强了特征复用
  • 相对判别器 (Relativistic Discriminator)：判别器预测"真实图像比假图像更真实"的相对值，而非绝对真伪。这使判别器不仅学习判断真实分布，还学习判断假分布
  • 激活前感知损失 (VGG before activation)：使用 VGG 网络激活层之前的特征计算感知损失，比激活后特征提供更强的亮度一致性和纹理恢复监督

关键创新：RRDB 无 BN 设计成为后续大量 SR 网络的基础模块；相对判别器思想被广泛采用。

性能特点：在 PIRM2018-SR Challenge Region 3 中获得第一名。视觉效果显著优于 SRGAN，纹理更真实自然。但存在过度生成伪影问题。

⚠️ ESRGAN 论文为 2018 年发表，arXiv 无 HTML 版本，图片仅见于 PDF。请参阅 PDF 原文 Figure 1-3 查看 RRDB 架构图和重建结果对比。

2. BSRGAN (2021) — Designing a Practical Degradation Model for Deep Blind SR

Degradation Model Blind SR Isotropic/Anisotropic Blur Camera ISP Noise

核心思想：设计更复杂、更实用的退化模型来训练盲超分网络，使其能处理真实世界多样退化。

退化模型设计：

  • 随机打乱退化顺序：模糊 + 下采样 + 噪声的组合顺序随机排列，覆盖更多退化空间
  • 各向同性/各向异性高斯模糊：不仅使用圆形高斯核，还使用椭圆高斯核模拟方向性模糊
  • 多种下采样：最近邻、双线性、双三次插值随机选择
  • 复合噪声：高斯噪声 + JPEG 压缩（不同质量因子）+ 相机传感器噪声（通过逆正向 ISP 管道模型生成）
关键创新：首次系统性地设计实用的退化模型，随机打乱退化顺序使模型覆盖更广的退化空间。相机 ISP 噪声模拟是突破性设计。ICCV 2021 发表。

性能特点：在合成和真实图像上都显著优于使用简单退化模型训练的 ESRGAN，验证了退化模型设计对盲 SR 的重要性。

⚠️ BSRGAN 论文为 ICCV 2021，arXiv 无 HTML 版本。请参阅 PDF 原文 Figure 1 查看退化模型对比图。

3. Real-ESRGAN (2021) — Training Real-World Blind SR with Pure Synthetic Data

Second-order Degradation U-Net Discriminator Spectral Normalization Pure Synthetic Data

核心思想：将 ESRGAN 扩展为实用化工具，通过高阶退化建模和改进的判别器，仅用纯合成数据训练即可处理真实世界复杂退化。

架构设计：

  • 二阶退化建模 (High-order Degradation)：将退化过程建模为两阶串联，每阶包含模糊+下采样+噪声+JPEG 压缩。二阶建模更好地模拟真实世界复杂退化链
  • 振铃和过冲伪影模拟：在合成过程中考虑振铃 (ringing) 和过冲 (overshoot) 伪影，使训练数据更贴近真实
  • U-Net 判别器 + 谱归一化：U-Net 结构提供像素级判别反馈，谱归一化稳定训练动态。比传统 PatchGAN 判别器更强大
关键创新：二阶退化建模是核心突破——通过串联两个退化阶段，模拟真实世界退化链（如先模糊再压缩再噪声）。纯合成数据训练消除了对真实配对数据的依赖。

性能特点：在各种真实数据集上视觉表现优于先前方法。提供高效实时合成训练对的实现。成为广泛使用的实用 SR 工具。

⚠️ Real-ESRGAN 为 Tech Report，arXiv 无 HTML 版本。请参阅 PDF 原文 Figure 2 查看高阶退化流程图。

4. StableSR (2023) — Exploiting Diffusion Prior for Real-World SR

Diffusion Prior T2I Model Time-aware Encoder Controllable Feature Wrapping Progressive Aggregation

核心思想：利用预训练的文本到图像 (T2I) 扩散模型中蕴含的丰富图像先验，用于盲超分任务。

架构设计：

  • Time-aware Encoder：将 LQ 图像编码为时间步感知的条件特征，注入扩散模型。不修改预训练 T2I 模型权重，保留其生成能力
  • Controllable Feature Wrapping：通过一个标量值控制质量和保真度的平衡——用户可在推理时调整，无需重训模型
  • Progressive Aggregation Sampling：渐进式聚合采样策略，克服预训练扩散模型固定分辨率限制，适应任意尺寸输入
关键创新：首次成功利用预训练 T2I 扩散模型先验做盲 SR，不修改预训练模型（保留生成先验），通过 time-aware encoder 注入条件。可控特征包装模块解决了扩散模型随机性与图像恢复确定性之间的矛盾。IJCV 2024 发表。

性能特点：在合成和真实基准上超越当时 SOTA。但需要多步采样（~50-200 步），推理速度慢。

5. DiffBIR (2023) — Towards Blind Image Restoration with Generative Diffusion Prior

Two-stage IRControlNet Region-adaptive Guidance LDM

核心思想：将盲图像恢复解耦为两阶段：1) 退化去除（去除与图像无关的退化内容）；2) 信息再生（用扩散先验生成丢失的图像内容）。

架构设计：

  • Stage 1 - 退化去除：使用恢复模块去除退化，获得高保真（但可能缺乏细节）的中间结果
  • Stage 2 - 信息再生：提出 IRControlNet，利用 LDM (Latent Diffusion Model) 的生成能力。基于特殊制作的条件图像（去除干扰噪声内容）训练，保证稳定生成
  • Region-adaptive Restoration Guidance：推理时可调节的去噪引导，无需重训模型即可平衡真实感和保真度
关键创新：两阶段解耦是核心设计——退化去除和信息再生各司其职，避免单阶段方法中保真度与真实感的冲突。IRControlNet 基于 ControlNet 思想但专为图像恢复设计。ECCV 2024 发表。

性能特点：在盲 SR、盲人脸恢复和盲图像去噪三个任务上均超越 SOTA。统一框架处理多种恢复任务是重要优势。

⚠️ DiffBIR 论文 arXiv HTML 图片不可直接下载。请参阅 arXiv HTML 版 或 GitHub 项目页 查看架构图。

6. SeeSR (2023) — Towards Semantics-Aware Real-World Image SR

Semantics-Aware DAPE Soft/Hard Prompts LR-to-Noise

核心思想：强退化会导致局部结构破坏，进而引发语义歧义。SeeSR 通过语义感知方法保持恢复图像的语义保真度。

架构设计：

  • DAPE (Degradation-Aware Prompt Extractor)：训练一个退化感知提示提取器，即使在高强退化下也能生成准确的语义提示
    • 硬语义提示 (Hard Prompts)：图像标签 (tags)，增强 T2I 模型的局部感知能力
    • 软语义提示 (Soft Prompts)：补充硬提示，提供额外的表征信息
  • LR 图像融入初始采样噪声：将 LQ 图像信息融入扩散采样的初始噪声中，减轻扩散模型生成过度随机细节的倾向
关键创新：首次在 Real-ISR 中显式引入语义理解——DAPE 提取的软/硬语义提示解决了退化导致语义歧义的问题。LR 融入初始噪声是减少扩散随机性的巧妙设计。CVPR 2024 发表。

性能特点：在语义保真度和细节真实感上优于先前方法。DAPE 的退化感知能力使其在强退化场景下仍能正确理解图像内容。

7. OSEDiff (2024) — One-Step Effective Diffusion Network for Real-ISR

One-Step LQ-as-Starting-Point LoRA VSD NeurIPS 2024

核心思想：LQ 图像本身包含丰富的恢复信息，可直接作为扩散起点，无需从随机噪声开始，从而实现单步推理。

架构设计：

  • LQ 图像作扩散起点：不同于从随机噪声开始的标准扩散，OSEDiff 直接将 LQ 图像（加噪）作为起点，消除随机性引入的不确定性
  • LoRA 微调：用可训练的 LoRA 层微调预训练扩散网络，适应复杂图像退化，不修改原始权重
  • 变分分数蒸馏 (VSD)：在潜空间进行 KL 散度正则化，确保单步扩散模型输出质量。使用预训练的多步扩散模型作为教师
关键创新：三个设计协同实现单步推理：① LQ 作起点消除不确定性；② LoRA 高效微调；③ VSD 潜空间蒸馏保证质量。单步推理达到多步方法（~50步）的质量。NeurIPS 2024 发表。

性能特点：客观指标和主观评价均可比或超越需要数十/数百步的方法。推理速度提升 50-200×。

8. 最新方法 (2025-2026)

8.1 ODTSR (2025-11) — One-Step Diffusion Transformer for Controllable Real-ISR

One-Step DiT Qwen-Image Noise-hybrid Visual Stream

核心思想：基于 Qwen-Image 的单步扩散 Transformer，同时兼顾保真度和可控性。引入双视觉流设计：一个流接收可调噪声的 LQ 图像（Control Noise），另一个接收一致噪声的 LQ 图像（Prior Noise），形成 Noise-hybrid Visual Stream (NVS)。通过 Fidelity-aware Adversarial Training (FAA) 增强可控性并实现单步推理。

关键创新：双流 NVS 设计将"控制信号"和"先验信号"分离，使模型既能控制保真度又能利用扩散先验。还能实现中文场景文字图像超分（STISR）的零样本可控性。

8.2 FRAMER (2025-12) — Frequency-Aligned Self-Distillation with Adaptive Modulation

Frequency-Aware Self-Distillation FFT CVPR 2026

核心思想：扩散模型在高频细节重建上不足，存在低频偏置和"先低频后高频"的层级结构。FRAMER 是一种即插即用的训练方案，通过频率对齐自蒸馏增强高频细节恢复。

  • 自蒸馏：最终层特征图作为教师，指导所有中间层（学生）
  • FFT 频率分解：将特征图通过 FFT mask 分解为低频 (LF) 和高频 (HF) 频段
  • 对比损失：低频用 Intra Contrastive Loss 稳定全局结构，高频用 Inter Contrastive Loss 锐化实例特定细节
  • 自适应调制：FAW 和 FAM 两个调制器重新加权每层 LF/HF 信号
关键创新：从频域视角理解和改进扩散 SR 模型，不改变骨干网络和推理流程。适用于 U-Net 和 DiT 架构（SD2、SD3）。CVPR 2026 接收。

8.3 SANA-SR (2026-05) — Efficient One-Step Diffusion with Compact Token Compression

One-Step Linear Attention 32× Compression Mobile Deployment

核心思想：Real-ISR 的瓶颈不在恢复先验不足，而在 token 冗余和高分辨率下昂贵的 token 交互。通过紧凑潜在表示和线性复杂度建模实现高效部署。

  • 32× 深度压缩自编码器：将压缩比提升到 32×，大幅减少 latent tokens，同时保留恢复相关结构和纹理
  • 线性注意力 DiT + LoRA：在线性复杂度下做 token 混合，LoRA 微调适配恢复任务
  • 部署指标：剪枝后 0.019s 推理，407.95G MACs，344M 参数——具备移动端部署潜力
关键创新：从部署效率角度重新思考 Real-ISR 架构。32× 压缩比 + 线性注意力 = 移动级实时 SR。证明了 token 冗余是核心瓶颈而非先验不足。

8.4 TEASR (2026-06) — Training-Efficient Any-Step Diffusion Transformer for Real-ISR

Any-Step Self-Adversarial Distillation 20B Parameters Single GPU

核心思想：在单一扩散模型内进行自对抗蒸馏，无需辅助教师模型或判别器，支持任意步数采样（1步到多步无缝切换）。

  • 自对抗蒸馏 (Self-Adversarial Distillation)：单一模型内的自蒸馏，消除对辅助教师和判别器的依赖
  • 时间步感知矫正策略：稳定跨噪声水平的单步生成
  • 双分支 DiT：解耦时间步条件，分离当前噪声状态和去噪目标
  • 20B 参数蒸馏：上述设计使 20B 参数扩散模型可在单 GPU 上蒸馏
关键创新：自对抗蒸馏消除了大模型蒸馏的内存瓶颈——20B 模型单 GPU 蒸馏是训练效率的突破。任意步数采样提供了速度与质量的灵活权衡。

🔄 范式转变分析

转变 1：GAN → Diffusion (2023)

驱动力：预训练 T2I 扩散模型（Stable Diffusion）蕴含丰富的自然图像先验，远超 GAN 的生成能力。
维度GAN 范式 (ESRGAN/Real-ESRGAN)Diffusion 范式 (StableSR/DiffBIR/SeeSR)
生成先验判别器学习的局部分布预训练 T2I 模型的全局自然图像先验
纹理质量较好但易出现伪影更真实自然的纹理
推理速度快 (单次前向)慢 (50-200 步采样)
保真度较高 (确定性映射)较低 (扩散随机性)
语义理解无SeeSR 引入语义提示

转变 2：多步 → 单步 (2024)

驱动力：多步扩散推理慢且存在随机性，与图像恢复的确定性需求矛盾。OSEDiff 开创单步 Real-ISR。
维度多步扩散 (StableSR/DiffBIR/SeeSR)单步扩散 (OSEDiff/SANA-SR)
推理步数50-200 步1 步
推理速度慢 (秒级)快 (0.019s ~ 0.1s)
随机性高 (从随机噪声开始)低 (LQ 作起点)
保真度较低较高
训练方式微调预训练模型蒸馏 + LoRA 微调

转变 3：U-Net → DiT (2025-2026)

驱动力：Diffusion Transformer (DiT) 架构在图像生成领域取代 U-Net，Real-ISR 也跟随这一趋势。
  • ODTSR：基于 Qwen-Image 的 DiT 架构，双流视觉设计
  • SANA-SR：线性注意力 DiT + 32× 压缩，面向移动端
  • TEASR：双分支 DiT，解耦时间步条件，支持 20B 参数
  • FRAMER：适用于 U-Net 和 DiT 的频率对齐训练方案

趋势 4：RL 后训练引入 (2025-2026)

FinPercep-RM (CVPR 2026, arXiv 2512.22647) 将 RLHF 引入 Real-ISR，设计细粒度奖励模型和协同进化课程，表明强化学习后训练正从视频生成/LLM 领域向图像恢复领域扩展。

💡 个人分析

  • 退化建模是 GAN 时代的核心竞争点：从 ESRGAN 的简单退化到 BSRGAN 的随机打乱退化再到 Real-ESRGAN 的二阶退化，退化建模的复杂度和真实性直接决定了模型的实用性。但退化建模存在天花板——真实世界退化是无限多样的，任何显式建模都无法完全覆盖。
  • 扩散先验是范式转变的关键：StableSR 首次利用预训练 T2I 扩散模型先验做 SR，这是从"学习退化映射"到"利用自然图像先验"的根本转变。扩散模型在数亿图像上学习到的先验远比 GAN 判别器学习到的局部分布丰富。DiffBIR 的两阶段解耦和 SeeSR 的语义感知进一步细化了这一范式。
  • 单步推理是实际落地的必须条件：OSEDiff 的突破性在于认识到"LQ 图像本身是最好的起点"——不需要从随机噪声开始，从而消除随机性并实现单步推理。SANA-SR (0.019s) 和 TEASR (任意步) 进一步推动效率边界。单步方法的成熟标志着 Real-ISR 从研究走向部署。
  • 语义理解是被低估的方向：SeeSR 的 DAPE 是唯一显式引入语义理解的工作。但 ODTSR 在中文 STISR 上的零样本能力表明，语义可控性是未来重要方向。结合多模态大模型的语义理解可能是下一步突破点。
  • 频率视角是新的优化角度：FRAMER 从频域视角分析扩散模型的不足（低频偏置、先低后高层级），并用 FFT 分解做自蒸馏。这是一个不改变架构即可提升性能的通用方案，可能被广泛采用。
  • DiT 架构迁移正在进行：2025-2026 年的新方法（ODTSR、SANA-SR、TEASR）全部采用 DiT 架构而非 U-Net。这与图像生成领域 DiT 取代 U-Net 的趋势一致。DiT 的扩展性更好，能支持更大模型（TEASR 的 20B 参数）。
  • 未来方向预测：
    • Flow Matching + SR：Flow Matching 在图像生成中已展现出优于 ODE-based diffusion 的潜力，但目前在 SR 领域应用还很少（arxiv 搜索结果多为不相关），是重要研究机会
    • RL 后训练：FinPercep-RM (CVPR 2026) 的出现表明 RL 后训练开始进入 Real-ISR，与视频生成和 LLM 领域趋势一致
    • 更大模型 + 更高效推理：TEASR 的 20B 参数蒸馏和 SANA-SR 的移动端部署代表了两个方向——更大模型提供更强先验，更高效推理实现部署
    • 视频 Real-ISR：DiffST (arXiv 2605.13182) 和 LiteVSR (arXiv 2606.09250) 开始将扩散 Real-ISR 扩展到视频领域，时空一致性是新的挑战
    • 任意尺度 SR：OmniScaleSR (arXiv 2512.04699) 和 OP4KSR (arXiv 2605.13457) 探索任意尺度和 4K SR，扩展应用场景
  • GAN 范式并未过时：Real-ESRGAN 至今仍是使用最广泛的实用 SR 工具——单次前向、速度快、部署简单。扩散方法在质量上更优但在速度上仍落后。未来的方向可能是 GAN 和 Diffusion 的融合——用扩散先验指导 GAN 训练，或用 GAN 做快速初始化再用扩散精化。
