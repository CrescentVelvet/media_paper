📌 技术方向概要

几何一致性 (Geometric Consistency) 是视频生成中的核心挑战：生成视频在时间维度上应保持 3D 场景结构的稳定性——物体不应变形或闪烁、透视关系应正确、运动应符合物理规律。当前视频扩散模型常出现几何不一致问题：物体在帧间闪烁/变形、透视错误、运动不自然。

调研发现：该方向正从架构修改注入 3D 先验转向强化学习约束 3D 一致性，从RGB 空间循环转向潜空间 3D 记忆，从单视角独立生成转向多视角点追踪监督。

🏗️ 技术方案详解

1. World-R1: RL 强化 3D 约束

来源：arXiv 2604.24764, ICML 2026, Microsoft, 2026-04-27

核心问题：视频基础模型视觉合成能力强，但频繁出现几何不一致。现有方法通过架构修改注入 3D 先验，计算成本高且限制可扩展性。

方案：

  • RL 框架：使用 Flow-GRPO（强化学习策略梯度方法适配 flow matching 模型）优化 3D 一致性
  • 3D 基础模型反馈：使用预训练 3D 基础模型提供反馈，惩罚几何不一致
  • VLM 反馈：视觉语言模型 (VLM) 评估结构连贯性
  • 纯文本数据集：引入专为世界模拟设计的纯文本数据集
  • 周期解耦训练：交替训练刚性几何一致性和动态场景流动性，平衡两者
  • 无架构修改：不改变基础模型架构，仅通过 RL 后训练对齐
关键创新：首个将 RL 用于 3D 一致性约束的工作。不修改架构 → 可在任何视频扩散模型上应用。周期解耦训练解决了"刚性几何 vs 动态运动"的矛盾。

2. Robust Dreamer: 潜空间 3D 高斯记忆

来源：arXiv 2605.30855, 2026-05-29

核心问题：自回归 (AR) 视频生成中长序列 rollout 出现灾难性漂移，原因有二：

  • Latent-RGB Cycling：生成的潜空间表示被反复解码为 RGB 再编码回来用于未来条件，每步都丢失信息
  • Error-free Hypothesis：训练时用干净记忆，推理时用预测（有误差）记忆，导致训练-推理差距

方案：

  • Latent Gaussian Memory：将扩散过程中的潜空间表示锚定到 3D 高斯原语 (Gaussian Primitives)，通过潜空间高斯泼溅 (Gaussian Splatting) 回忆——避免 RGB 中转的信息损失
  • Deviation Learning：通过一步近似合成 rollout 引起的潜空间偏差，按自回归阶段和去噪时间步存储到 Dynamic Deviation Archive，训练时注入到历史记忆中
  • 训练-推理对齐：让模型在训练时就面对"被污染"的记忆状态，学会内部纠错
核心贡献：在潜空间而非 RGB 空间维护 3D 记忆，从根源上解决了 Latent-RGB Cycling 的信息损失。偏差学习是首个显式处理训练-推理记忆差距的方法。

3. MVTrack4Gen: 多视角点追踪监督

来源：arXiv 2606.26087, KAIST + Sony, 2026-06-24

核心问题：仅依赖相机条件的方法视觉质量高但几何/运动一致性差；基于显式 3D 表示的方法受限于重建模块精度。

方案：

  • 关键发现：特定注意力层的 query-key 特征在几何对应位置跨视图跨时间形成强对应信号，这些对应的不对齐导致运动不一致
  • Auxiliary Multi-View Tracking Head：将这些注意力特征路由到辅助多视角追踪头，与扩散模型联合训练点追踪目标
  • 显式强化对应：通过点追踪目标显式强化跨视图运动对应
  • 即插即用：可增强现有相机条件模型，不需要重建模块
创新点：利用注意力层中已存在的几何对应信号，通过辅助任务 (点追踪) 显式强化。这是"发现-利用"而非"注入"外部 3D 先验的新范式。

📊 与相机可控性的交叉

几何一致性与相机可控性密切相关——相机控制要求生成视频遵循指定轨迹，而几何一致性确保在轨迹变化时场景结构稳定。几个交叉点：

  • Track2View (相机控制) + MVTrack4Gen (几何一致)：两者都使用点追踪，前者用作条件信号，后者用作监督信号
  • Geo-Align (相机控制) + World-R1 (几何一致)：两者都使用 RL 和 3D 基础模型反馈，前者约束相机轨迹准确性，后者约束 3D 结构一致性
  • Robust Dreamer 的 Latent Gaussian Memory：同时服务几何一致性（3D 锚定）和自回归稳定性（避免 RGB 循环），对 AR 视频生成场景（如世界模拟）有重要意义

💡 个人分析

  • RL 范式双重应用：World-R1 和 Geo-Align 都在用 RL 解决视频生成问题——前者约束 3D 一致性，后者约束相机轨迹精度。RL 后训练正在成为视频生成的通用优化工具，类似 LLM 中的 RLHF 范式
  • 潜空间 3D 记忆的突破：Robust Dreamer 的 Latent Gaussian Memory 是重要创新。传统方法在 RGB 空间维护记忆，每步 VAE 编解码都损失信息。在潜空间直接用高斯原语表示和回忆，避免了这一衰减，对长序列 AR 视频生成尤为关键
  • 注意力层中已蕴含几何信号：MVTrack4Gen 发现注意力层特征已包含几何对应信号，这是重要洞察。不需要从外部注入 3D 先验，而是发现并强化模型内部已存在的几何理解能力
  • 训练-推理差距是被忽视的问题：Robust Dreamer 的 error-free hypothesis 分析指出了一个被忽视的训练-推理差距。在 AR 视频生成中，训练时记忆是干净的，推理时记忆是模型自己生成的（有误差）。偏差学习显式处理这一问题
  • 世界模拟愿景：World-R1 引入"世界模拟纯文本数据集"，Robust Dreamer 在 ScanNet/DL3DV/OmniWorldGame 上测试，都指向视频生成 → 世界模拟的长期愿景。几何一致性是世界模拟的基础能力
  • 刚性 vs 动态的矛盾：World-R1 的周期解耦训练显式处理这一矛盾——过于强调几何一致性会导致场景僵硬，过于强调动态性会导致结构不稳定。这是视频生成中独有的挑战
  • 与 Cosmos 3 的关系：Cosmos 3 作为全模态世界模型，其 3D 一致性通过统一 MoT 架构和大规模数据隐式实现。而本调研的三篇论文提供了更显式、更有针对性的 3D 一致性方案
  • 开源生态：World-R1 有 Microsoft 开源代码 (github.com/microsoft/World-R1)，Robust Dreamer 和 MVTrack4Gen 提供项目页面。这些工作为社区提供了可直接使用的工具

阶段方法代表工作

核心损失函数

  • World-R1：L = L_GRPO + R_geometric，几何奖励基于 3D 一致性指标
  • Robust Dreamer：L = L_diffusion + λ·L_gaussian_memory（潜空间高斯记忆对齐）
  • MVTrack4Gen：L = L_diffusion + λ·L_track（attention 层内点追踪损失）

训练策略

  • World-R1：SFT 基线 → GRPO 几何对齐
  • Robust Dreamer：在潜空间维护高斯记忆，跨帧对齐
  • MVTrack4Gen：在 attention 层嵌入点追踪监督，端到端学习
演进趋势：从架构层面改 → RL 约束 → 潜空间记忆 → 注意力层内直接监督。越来越深入模型内部，从外部正则化到内部结构化监督。几何一致性是视频生成从“美学好看”到“物理正确”的关键。
