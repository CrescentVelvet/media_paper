3D Gaussian Splatting: 基础技术与改进综述

📄 综述覆盖: 2308.04079 → 2402.14650 → 2403.11134 📅 2023-08 ~ 2026 🏢 INRIA / 多机构 📂 综述

📋 基础: 3D Gaussian Splatting for Real-Time Radiance Field Rendering

Novel View Synthesis Explicit Representation Differentiable Rendering Real-Time SIGGRAPH 2023

arXiv: 2308.04079 | 发表: ACM Transactions on Graphics, SIGGRAPH 2023 | 作者: Kerbl, Kopanas, Leimkühler, Drettakis (INRIA)

3D Gaussian Splatting (3DGS) 是神经渲染领域的范式转变。与 NeRF 使用隐式神经网络表示场景不同，3DGS 用数百万个可学习的 3D 高斯椭球体显式表示场景，通过可微光栅化（differentiable rasterization）将高斯投影到图像平面进行渲染。

核心贡献：首次实现了 1080p 分辨率下 30+ FPS 实时新视角合成，同时训练时间与 NeRF 相当甚至更短，渲染质量超越 NeRF。

三大关键要素

  • 3D 高斯场景表示：从 SfM 稀疏点云初始化，每个高斯有位置、协方差（各向异性）、不透明度、球谐系数（颜色）等属性
  • 交错优化/密度控制：自适应密度控制——根据梯度信号自动克隆、分裂或删除高斯，优化各向异性协方差以精确拟合场景
  • 快速可见性感知渲染：开发支持各向异性 splatting 的快速光栅化算法，既加速训练又允许实时渲染

🔧 改进工作综述

1. GaussianPro: 渐进传播策略 (arXiv: 2402.14650)

Progressive Propagation MVS-inspired Large-scale

问题：3DGS 严重依赖 SfM 初始化的点云。在大规模场景的无纹理区域，SfM 无法产生足够点云，导致 3DGS 优化困难和低质量渲染。

方法：受经典多视角立体视觉（MVS）启发，采用渐进传播策略指导高斯的密度化。利用已重建几何的先验和 patch matching 技术生成位置和朝向准确的新高斯，替代 3DGS 简单的分裂和克隆策略。

结果：在 Waymo 数据集上 PSNR 提升 1.15dB，大规模场景改善显著。

2. 3DGS-MCMC: MCMC 采样替代梯度密度化

MCMC Density Control

问题：3DGS 原始的密度控制（分裂/克隆/删除）基于启发式梯度阈值，可能导致高斯分布不合理。

方法：将高斯的位置优化重新表述为马尔可夫链蒙特卡洛（MCMC）采样问题，通过 Metropolis-Hastings 接受/拒绝准则来控制高斯的增减，理论上更 principled。

影响：3DGS-MCMC 已成为后续多个工作（如 PixelGS、压缩方法等）的标准基线之一。

3. Mip-Splatting: 解决锯齿问题

Anti-aliasing Scale-aware

问题：标准 3DGS 在不同分辨率渲染时存在锯齿和模糊问题，因为没有考虑采样面积和高斯大小的关系。

方法：引入额外的 2D 滤波器（类似 EWA splatting），根据像素采样面积调整高斯的 2D 投影大小，实现尺度一致的渲染。

4. 2D Gaussian Splatting (2DGS)

Surface Reconstruction Geometry

问题：3DGS 的各向异性 3D 高斯可能退化为扁平的"煎饼"状，导致几何不准确。

方法：将 3D 高斯退化为 2D 椭圆盘（disk），直接表示表面，改善几何重建质量。Long-LRM 已探索与 2D GS 的兼容性。

5. SuGaR: 表面对齐的高斯

Mesh Extraction Surface Alignment

方法：通过正则化使高斯与场景表面对齐，便于从高斯中提取高质量网格。解决了 3DGS 难以导出网格的问题。

6. Scaffold-GS: 锚点引导的高斯

Anchor-based Structured

方法：引入锚点机制，每个锚点关联一组高斯，通过可学习的偏移生成高斯属性。相比 3DGS 的自由增长策略，Scaffold-GS 提供了更有组织的结构，减少冗余高斯。

💡 个人分析

  • 3DGS 是 NeRF 的真正继任者：2023 年发布以来，3DGS 迅速成为新视角合成的主流方法。实时渲染、显式表示、可编辑性三大优势使其在实际应用中远超 NeRF。
  • 改进工作的分层：3DGS 改进可分为：(1) 渲染质量（Mip-Splatting, 2DGS）；(2) 几何质量（SuGaR, 2DGS）；(3) 效率（压缩方法, Scaffold-GS）；(4) 规模（GaussianPro）。其中密度控制是最核心的改进方向。
  • 从前馈到优化的光谱：3DGS 本身需要场景级优化（~30 分钟），而 GS-LRM/Long-LRM 等前馈方法实现了秒级重建。但优化方法在质量上仍有优势，两者是互补关系。
  • 与 VGGT 的关系：VGGT 可以为 3DGS 提供高质量的初始化点云和位姿，解决 3DGS 依赖 SfM 的痛点。两者结合是自然的方向。
  • 4D 扩展：动态 3DGS（4D Gaussians）是自然扩展方向，已在动态场景重建和追踪中展现潜力。
  • 生态成熟度：截至 2026 年，3DGS 已有 200+ 改进工作、多个综述论文、标准化基准。生态成熟度远超 NeRF 同期。

3DGS 的核心渲染管线:

  • 初始化: 从 SfM 稀疏点云初始化 3D 高斯椭球体(位置=点云点, 协方差=SfM 不确定性)
  • 可微光栅化: 3D 高斯→2D 投影→alpha 混合→像素颜色
  • 优化: 反向传播渲染损失→更新高斯参数(位置、协方差、不透明度、球谐系数)
  • 密度控制: 基于梯度大小自动克隆/分裂/删除高斯
  • 渲染: 按深度排序→可见性剔除→分块光栅化

💡 核心创新点

创新 1: 实时可微光栅化
首次为 3D 高斯设计了可微光栅化算法, 支持各向异性 splatting, 既加速训练又允许实时渲染。这是区别于 NeRF 的核心突破——NeRF 需要逐点 MLP 前向传播, 而 3DGS 直接光栅化。
创新 2: 自适应密度控制
基于位置梯度的大小自动克隆(小梯度+高位置不确定性)或分裂(大梯度+大尺寸), 删除低不透明度的高斯。这种自适应策略使场景表示在需要的地方密集, 不需要的地方稀疏。
创新 3: 各向异性 3D 高斯
每个高斯有独立的 3D 协方差矩阵(3 个缩放参数 + 4 个四元数参数), 可以退化为各向异性的椭球体, 精确拟合场景几何, 比各向同性的 NeRF 点表示更高效。
创新 4: 球谐系数颜色表示
使用球谐函数(SH)表示视角相关颜色, 每个 Gaussian 存储 SH 系数(通常到 3 阶), 实现视角相关的颜色变化, 处理镜面反射和透明材质。
创新 5: 显式表示 vs 隐式表示
3DGS 用显式的高斯参数(位置、大小、颜色)表示场景, 天然支持编辑(移动、删除、融合高斯), 而 NeRF 的隐式表示难以定位和修改特定区域。

🏋️ 训练策略

3DGS 优化流程

  • 初始化: 从 SfM (COLMAP) 稀疏点云初始化高斯位置, 协方差从 SfM 不确定性估计
  • 交错优化: 每个迭代: 渲染→计算损失→反向传播→更新参数→密度控制
  • 渐进式训练: 初始高斯少(~100K), 逐步增加到数百万
  • 学习率调度: 位置使用指数衰减学习率, 其他参数使用常数学习率

改进训练策略

  • 3DGS-MCMC: 将位置优化重新表述为 MCMC 采样, 使用 Metropolis-Hastings 准则控制高斯增减, 更 principled
  • GaussianPro: 渐进传播策略, 受 MVS 启发, 用 patch matching 生成新高斯
  • Scaffold-GS: 锚点引导, 先训练锚点再生成高斯, 更有组织

🔬 损失函数

3DGS 标准损失

  • 渲染损失: L_render = L1(I_render, I_gt) + λ · L_D-SSIM(I_render, I_gt)
  • L1 距离提供像素级保真, D-SSIM 提供结构相似性
  • λ 通常设为 0.2, 平衡像素和结构损失

正则化项 (改进工作)

  • Mip-Splatting: 2D 滤波正则化, 防止高斯在不同尺度下的锯齿
  • Scaffold-GS: 锚点正则化, 约束高斯与锚点的偏移
  • 3DGS-MCMC: 无显式正则化, 通过 MCMC 接受/拒绝准则隐式约束
  • 2DGS: 表面法向量一致性正则化
3DGS 训练的关键特点:没有预训练阶段, 完全从 SfM 点云开始场景级优化(~30分钟)。这与前馈方法(VGGT, GS-LRM)形成对比, 后者通过大规模预训练实现秒级重建但质量稍低。

📐 算法框图

3DGS Pipeline 概览：从 SfM 点云初始化 → 高斯参数优化 → 实时可微渲染

核心 Pipeline：
  • 输入：多视角图像 + SfM 点云（COLMAP）
  • 初始化：SfM 稀疏点云 → 初始化 3D 高斯（位置=点坐标，颜色=SH 初始化）
  • 优化：可微光栅化渲染 → 与 GT 图像比 L1 + D-SSIM → 反向传播更新高斯参数
  • 自适应密度控制：梯度大的区域分裂/克隆，小的高斯删除，周期性密集化和剪枝
  • 输出：百万级 3D 高斯 → 实时渲染（>100 FPS @ 1080p）

高斯参数（每个高斯 59 维）：

  • 位置 μ ∈ R³ (3D)
  • 不透明度 α ∈ [0,1] (1D)
  • 协方差 Σ ∈ R^{3×3} (分解为缩放 S ∈ R³ + 旋转 R ∈ SO(3)，共 6D)
  • 球谐系数 SH (3阶 = 48D，用于视角依赖颜色)

渲染公式：C(x) = Σᵢ αᵢ · cᵢ · Tᵢ，其中 Tᵢ = ∏ⱼ₍ⱼ

💡 核心创新点

创新 1：3D 高斯场景表示 — 用各向异性 3D 高斯椭球体作为场景表示，兼具显式（可编辑）和隐式（高质量）的优点，超越 NeRF 的隐式表示。
创新 2：可微光栅化 — 将 3D 高斯投影到 2D 后进行 tile-based 排序和 alpha 混合，全流程可微，支持端到端梯度优化。
创新 3：自适应密度控制 — 根据位置梯度动态分裂/克隆/删除高斯，高梯度区域密集化，低贡献区域剪枝，自动调节场景表示密度。
创新 4：实时渲染 — >100 FPS @ 1080p，比 NeRF 快 100×以上，达到实时应用标准。
创新 5：球谐颜色表示 — 用球谐函数表示视角依赖颜色，从不同角度看到不同颜色，提升真实感。

🏋️ 训练策略

3DGS 优化流程

  • 初始化：SfM 稀疏点云初始化高斯位置（每个点 = 一个高斯）
  • 优化循环：每次迭代 → 随机选视角 → 可微渲染 → 计算 loss → 反向传播 → 更新高斯参数
  • 密度控制周期：每 100 次迭代执行一次自适应密集化/剪枝
  • 训练时长：约 30-60 分钟/场景（30k 次迭代）

改进工作的训练策略

  • Mip-Splatting：引入抗锯齿滤波器，解决不同分辨率下的 aliasing
  • Scaffold-GS：Anchor-based 结构，从锚点生长高斯，更结构化
  • 3DGS-MCMC：将优化视为 MCMC 采样，避免自适应控制的超参敏感性
  • 2DGS：退化为 2D 盘片，更好的表面几何质量
  • SuGaR：提取网格，高斯贴合表面

📐 损失函数

3DGS 原始损失

  • 组合损失：L = (1 - λ)·L1 + λ·L_D-SSIM，默认 λ=0.2
  • L1：像素级绝对误差，保证基础重建质量
  • D-SSIM：结构不相似性，提升感知质量
