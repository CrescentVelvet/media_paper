📌 论文概要

Wan-R1 探索了如何通过可验证强化学习提升视频生成模型的空间推理和多步规划能力。研究发现，视频生成模型虽能产生视觉连贯内容，但在需要空间推理和多步规划的任务上表现不佳。RL 是提升泛化能力的途径，但其效果取决于奖励设计。

核心发现：多模态奖励模型在此场景下灾难性失败。作者设计了基于客观任务指标的可验证奖励函数，证明可验证奖励是稳定训练的关键，而多模态奖励模型会导致退化解。

🏗️ 模型架构与方法

RL 方法：GRPO 适配 Flow-based 视频模型

将 Group Relative Policy Optimization (GRPO) 适配到 flow-based 视频模型：

  • 基于 flow matching 的视频生成模型作为策略网络
  • RL 训练而非传统的扩散/flow matching 训练
  • 用于迷宫求解和机器人导航任务

奖励设计

任务奖励类型设计
结构化游戏环境（迷宫）多组件轨迹奖励基于客观任务指标的多分量奖励
机器人导航Embedding-level 可验证奖励基于嵌入空间的可验证奖励函数

关键发现

  • 多模态奖励模型失败：使用多模态奖励模型作为 RL 奖励会导致灾难性失败，模型产生退化解
  • 可验证奖励关键：基于客观任务指标的可验证奖励是稳定训练的必要条件
  • 泛化提升：RL fine-tuning 配合可验证奖励显著提升泛化能力

🎯 关键特性

  • 首次系统研究视频推理的奖励设计：系统分析了不同奖励类型在视频推理 RL 中的效果
  • 可验证奖励框架：为结构化环境和机器人导航分别设计了可验证奖励
  • GRPO 适配 flow 模型：首次将 GRPO 适配到 flow-based 视频生成模型

💡 个人分析

核心贡献

  • 开创性方向：Wan-R1 是首个系统研究视频生成模型推理能力提升的工作，将 RL 从 LLM 领域扩展到视频生成
  • 奖励设计洞察："多模态奖励模型灾难性失败"这一发现非常重要，说明视频推理 RL 不能简单照搬 LLM 的奖励策略
  • 可验证奖励范式：为视频推理 RL 提供了可行的奖励设计范式，类似于数学推理中可验证奖励的成功
  • 显著提升：避障任务 +51.4% 的提升非常显著，证明 RL 在视频生成中的潜力

📐 算法框图 / 💡 创新 / 🎯 应用 / 🏋️ 训练 / 📊 数据 / 📐 损失 / 🔄 演进

Wan-R1 架构：基于流匹配视频模型 + GRPO 强化学习 + 可验证奖励

核心创新

  • GRPO 适配视频模型：将 Group Relative Policy Optimization 从 LLM 迁移到 flow-based 视频模型
  • 可验证奖励函数：基于客观任务指标的奖励，而非多模态奖励模型（后者灾难性失败）
  • 多组件轨迹奖励：迷宫任务的结构化奖励
  • 嵌入级可验证奖励：机器人导航任务的 embedding 空间奖励

训练策略

  • SFT 基线模型 → GRPO 强化学习微调
  • 任务：迷宫求解、机器人导航
  • 多模态奖励模型对比实验 → 确认可验证奖励的必要性

损失函数

  • GRPO 损失：L = -E[min(r_t/clip(r_t, 1-ε, 1+ε))·A] - β·KL
  • r_t 为策略比率，A 为组相对优势
  • 可验证奖励：R = R_exact_match + R_trajectory + R_collision
演进：Wan-R1 代表了视频生成从“美学优化”到“推理优化”的转变。多模态奖励模型的失败说明 LLM 领域的 RM 不能直接迁移，可验证奖励是更可靠的方向。3D 迷宫 +29.1% exact match, 陷阱避兔 +51.4%。
