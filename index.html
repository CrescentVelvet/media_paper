1. Wan2.1: Open and Advanced Large-Scale Video Generative Models

📄 arXiv: 2503.20314 📅 2025-03-26 🏢 Alibaba / Wan-AI Team
DiT T2V I2V Wan-VAE Scaling Law 1.3B/14B

开源视频生成基础模型套件，14B 模型在多个基准超越闭源方案。1.3B 模型仅需 8.19GB VRAM，首个支持中英双语文字生成的视频模型。Wan-VAE 的 3D 因果设计成为后续工作的基础（包括 Cosmos 3 也采用）。

📖 阅读笔记 📄 arXiv 💻 GitHub

2. Wan2.2: Major Upgrade with MoE Architecture

📄 arXiv: (共享 2503.20314) 📅 2025-07-28 发布 🏢 Alibaba / Wan-AI Team
MoE 720P@24fps S2V Animate A14B/5B

Wan2.1 的重大升级：引入 MoE 架构到视频扩散模型（跨时间步专家分离），训练数据增加 65.6%(图像)/83.2%(视频)。扩展到语音驱动视频 (S2V) 和角色动画 (Animate)。TI2V-5B 支持 720P@24fps 在消费级 GPU 运行。

3. VACE: All-in-One Video Creation and Editing

📄 arXiv: 2503.07598 📅 2025-03-10 🏢 Alibaba / ali-vilab
Video Editing VCU Context Adapter Wan2.1 配套

一体化视频创作与编辑框架，通过 Video Condition Unit (VCU) 统一接口和 Context Adapter 将多种视频任务整合到一个模型中。支持参考视频生成、视频编辑、遮罩编辑等，性能对标专用模型。

4. Wan-Move: Motion-controllable Video Generation

📄 arXiv: 2512.08765 📅 2025-12-09 🏆 NeurIPS 2025 🏢 Alibaba / ali-vilab + 清华+港大
Motion Control Latent Trajectory Guidance MoveBench Wan-I2V-14B

运动控制视频生成框架，通过 Latent Trajectory Guidance 实现精细 point-level 运动控制，无需架构修改直接集成到 Wan-I2V-14B。运动可控性媲美 Kling 1.5 Pro Motion Brush。发布 MoveBench 评测基准。

📖 阅读笔记 📄 arXiv 💻 GitHub

5. Wan-R1: Verifiable-Reinforcement Learning for Video Reasoning

📄 arXiv: 2603.27866 📅 2026-03-29 🏢 中科院自动化所
Video Reasoning RL / GRPO Verifiable Rewards Maze Solving

首个系统研究视频生成模型推理能力的工作。将 GRPO 适配到 flow-based 视频模型，发现多模态奖励模型灾难性失败，设计可验证奖励函数实现稳定训练。迷宫求解 +29.1%，避障 +51.4%（vs SFT 基线）。

6. Wan-Streamer v0.1: End-to-end Real-time Interactive Foundation Models

📄 arXiv: 2606.25041 📅 2026-06-23 🏢 Alibaba / Wan-AI Team
Real-time Full-Duplex End-to-End Audio-Visual ~200ms Latency

原生流式端到端交互式基础模型，单一 Transformer 统一处理语言/音频/视频的输入输出。模型侧延迟 ~200ms，总延迟 ~550ms。不依赖外部 ASR/TTS/动画模块。唯一同时具备视频感知、视频输出、全双工、端到端、亚秒响应的模型。

7. Kandinsky 5.0: 图像与视频生成基础模型家族

📄 arXiv: 2511.14993 📅 2025-11-19 🏢 Kandinsky Lab AI (俄罗斯)
Video Generation DiT T2V / I2V SFT + RL 后训练 2B / 6B / 19B

8. HunyuanVideo 1.5: 轻量级高性能视频生成模型

📄 arXiv: 2511.18870 📅 2025-11-24 🏢 Tencent Hunyuan
8.3B Parameters SSTA Consumer GPU Step Distillation Muon Optimizer

仅 8.3B 参数达到开源视频生成 SOTA，消费级 GPU 可运行。SSTA (选择性+滑动窗口注意力) 是核心架构创新。Step Distillation 实现 RTX 4090 上 75 秒生成。工程完备度首屈一指。

9. LTX-Video / LTX-2: 实时视频与音视频联合生成

📄 arXiv: 2501.00103 / 2601.03233 📅 2024-12 ~ 2026-01 🏢 Lightricks
Realtime Video Audio-Visual 1:192 Compression 4K@50fps Modality-CFG

LTX-Video: 1:192 极高压缩率 + VAE 解码器去噪，2 秒生成 5 秒视频 (H100)。LTX-2: 首个开源 DiT 音视频联合基础模型，14B 视频 + 5B 音频双流，支持 4K@50fps + 同步音频。

10. 视频生成中的相机可控性技术方向

📄 综述: TriMotion / Track2View / Geo-Align 📅 2026-05 ~ 2026-06 📂 Camera Control
Camera Control Multi-Modal 3D Point Tracks RL Alignment ECCV 2026

💡 总结与思考

  • 开源生态引领：Wan 系列从 2.1 到 2.2 持续推动视频生成开源生态，已被 NVIDIA Cosmos 3、众多学术论文和社区项目采用
  • 架构创新：Wan-VAE (3D 因果 VAE) → MoE DiT → Block-causal Transformer，每个版本都有实质架构创新
  • 能力维度扩展：从基础 T2V/I2V 扩展到编辑 (VACE)、运动控制 (Move)、推理 (R1)、实时交互 (Streamer)
  • 消费级友好：1.3B 模型 8.19GB VRAM、TI2V-5B 消费级 720P，持续降低使用门槛
  • Wan2.3+ 不存在：截至 2026-06-25，没有 Wan2.3、Wan2.4、Wan2.5、Wan2.6、Wan2.7 的 arXiv 论文或 GitHub 仓库
  • 横向对比结论：HunyuanVideo 1.5 (8.3B SSTA) 在轻量+质量间取得最佳平衡；LTX-2 在音视频联合和 4K 方面领先；Kandinsky 5.0 提供最宽谱系 (2B~19B)
  • 技术方向趋势：RL 后训练正在成为通用优化工具（Wan-R1 推理、Geo-Align 相机控制、World-R1 几何一致性）；潜空间优化（LTX 1:192 压缩、Robust Dreamer 潜空间高斯记忆）是效率关键
  • 未来方向：Wan-Streamer v0.1 (192p) 暗示高分辨率实时交互是下一步；Wan-R1 的视频推理方向有待拓展到自然场景；相机可控性+几何一致性融合是世界模拟的基础
